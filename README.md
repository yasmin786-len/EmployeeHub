# EmployeeHub

# 👨‍💼 Employee Management System

A full-stack Employee Management System built with **Spring Boot 3**, **Java 21**, **React 18**, **Vite**, and **MySQL**. It enables efficient employee management through a modern and responsive web interface.

## 🚀 Features

- Employee CRUD Operations
- Search Employees
- Form Validation
- RESTful APIs
- Global Exception Handling
- Responsive UI

## 🛠️ Tech Stack

**Frontend**
- React 18
- Vite
- Axios
- CSS

**Backend**
- Java 21
- Spring Boot 3
- Spring Data JPA
- Hibernate

**Database**
- MySQL

## ▶️ Run Locally

### Backend
```bash
cd employee-management-backend
mvn clean install
mvn spring-boot:run
```

### Frontend
```bash
cd employee-management-frontend
npm install
npm run dev
```

Open: **http://localhost:5173**

## 📌 API Base URL

```
http://localhost:8080/api/employees
```
